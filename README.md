### Installing
 
```bash
make install
```

Reload GNOME. It will then register in your list of extensions. (Log off...)

```bash
gnome-extensions enable gnomebtcextension@daniel.gomes.digital
```
<br/><br/>
There is an ```uninstall``` block in the Makefile as well.
